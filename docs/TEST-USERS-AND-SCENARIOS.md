## Fake Test Documents

Located in `docs/test-documents/`. Upload these during registration or from the profile page.

| File | Use For | Format |
|------|---------|--------|
| `profile-photo.svg` | Profile picture upload | SVG (avatar illustration) |
| `national-id.svg` | National ID / Passport | SVG (ID card layout) |
| `proof-of-address.svg` | Proof of Address / Utility Bill | SVG (CEB electricity bill) |
| `insurance-card.svg` | Health Insurance Card | SVG (Swan Life insurance card) |
| `medical-degree.svg` | Medical Degree, Nursing Degree, Pharmacy Degree, Lab Degree | SVG (university diploma) |
| `professional-license.svg` | Professional License (any type) | SVG (Medical Council license) |
| `police-clearance.svg` | Police Clearance Certificate | SVG (character certificate) |
| `work-certificate.svg` | Work Certificate, Registration Certificate, Employment Proof, any other | SVG (hospital employment letter) |

> **Tip:** The signup form accepts `.pdf, .jpg, .jpeg, .png` and `.svg` files. All test documents are SVG format and work for any document slot.
